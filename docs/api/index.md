# API Reference

Welcome to the Note Sync Now API documentation.

## Available APIs

| API | Description |
|-----|-------------|
| [WebSocket API](./websocket-api) | Real-time synchronization protocol |
| [REST API](./rest-api) | HTTP endpoints for health checks and stats |

---

## WebSocket API

The WebSocket API is the primary interface for real-time note synchronization.

### Key Features

- Real-time bidirectional communication
- Room-based synchronization
- Automatic reconnection
- Chunked transfer for large content

[View WebSocket API Documentation →](./websocket-api)

---

## REST API

The REST API provides health checks and monitoring endpoints.

### Key Features

- Health status endpoint
- Server statistics
- Persistence status
